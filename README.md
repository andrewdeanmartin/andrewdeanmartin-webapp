# Andrew Dean Martin - Professional Portfolio

A modern, responsive personal website showcasing unique abilities as an AI Delivery Leader at PwC.

## Quick Start

**Option 1:** Open in browser:
```bash
open site/index.html
```

**Option 2:** Local server:
```bash
cd site
python3 -m http.server 8080
```
Then visit `http://localhost:8080`

## Files Included

- `index.html` - Main website structure
- `styles.css` - Design system, animations, demo styles
- `script.js` - Interactive elements and scroll effects
- `prompt-demo.js` - Prompt Refinement demo (Demos → Prompt Engineering)
- `space-invaders.js` - Easter egg
- `LINKEDIN-PROFILE.md` - Ready-to-use LinkedIn content

## Features

- **Navy/Copper Design** - Fraunces + IBM Plex Sans
- **Responsive Layout** - Desktop and mobile (including iPhone)
- **Live Demos** — Agent Pipeline, Prompt Engineering (interactive), Codebase Assessment
- **Prompt Refinement Demo** — Drag/tap flow, PC panels, copy-to-clipboard
- **Accessibility** - Semantic HTML, ARIA, keyboard support

## LinkedIn Content

The `LINKEDIN-PROFILE.md` file contains:
- Multiple headline options (under 120 characters)

- Experience entry with achievement bullets
- Skills to add/emphasize
- Tagline options

## Customization

### Update Contact Links
Edit the contact section in `index.html`:
```html
<a href="https://linkedin.com/in/YOUR-PROFILE" ...>
<a href="mailto:YOUR-EMAIL" ...>
```

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --color-primary: #6366f1;
    --color-accent: #22d3ee;
    /* ... */
}
```
